# Logo Setup Instructions

## Replace the Logo Image

To use your new logo, please follow these steps:

1. **Save your logo image** as `logo.png` in the `/public/images/` directory
2. **Recommended specifications:**
   - Format: PNG (with transparent background)
   - Size: 48x48 pixels minimum (or higher resolution for crisp display)
   - The logo should be circular as described

## Current Logo Implementation

The logo is now integrated in:
- **Header**: Shows on all pages in the top navigation
- **Footer**: Shows in the company information section
- **Mobile**: Responsive sizing for mobile devices

## Logo Features

- **Responsive**: Automatically scales for different screen sizes
- **Optimized**: Uses Next.js Image component for performance
- **Accessible**: Includes proper alt text for screen readers
- **Priority Loading**: Logo loads immediately for better UX

## File Location

```
/public/images/logo.png
```

Once you replace the placeholder file with your actual logo, it will appear throughout the website!
